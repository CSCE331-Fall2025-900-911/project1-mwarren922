Resi OS Migration:

To replace Centos 7, which was being used by Resi Media's x86 encoders, I created a custom and lightweight distribution of Debian 12, fine tuned for video encoding/decoding Using Debian's live-build tool. I implemented git-like version control on this OS, using OSTree for seamless deployment of updates.

I also developed and implemeted a live-migration tool for porting systems from CentOS7 to Debian 12 without any user interaction.
